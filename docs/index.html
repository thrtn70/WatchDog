Configuration

Most games work automatically.

A few need a quick config step.

CS2 (Counter-Strike 2)

CS2 requires a Game State Integration config file so WatchDog can detect kills, rounds, and aces. Copy the template to your CS2 config directory:

Copy-Item cfg\gamestate_integration_watchdog.cfg "C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg\"

Restart CS2 after placing the file.

✓

Valorant, Overwatch 2, and Rainbow Six Siege work automatically — no setup required. WatchDog reads game logs and the Riot Client lockfile to detect events.

Discord Sharing

Share clips to any Discord channel via webhook:

1. In Discord: Server Settings → Integrations → Webhooks → New Webhook → Copy URL
2. In WatchDog: Settings → Discord → Paste webhook URL → Save
3. Right-click any clip → "Share to Discord"

Troubleshooting

Common questions.

Click "More info" then "Run anyway." SmartScreen flags unsigned apps — WatchDog isn't code-signed yet. The installer and app are open-source; you can inspect the full source on GitHub.
Game capture may take 30-60 seconds to hook into the game process. Wait for the hook to attach, or switch to desktop capture mode in Settings. Anti-cheat games may take longer.
WatchDog recognizes 100+ games by process name. If yours isn't detected, use desktop capture mode (always-on). Custom game entries are planned for a future release.
1. Verify gamestate_integration_watchdog.cfg is in your CS2 cfg directory.
2. Restart CS2 after placing the file.
3. Check that recording mode is set to Highlights in Settings > Capture.
Another app (Game Bar, Discord, GeForce Experience) may have registered F9. Change the hotkey in Settings > Hotkeys, or use the system tray right-click menu to save clips.
Set up a webhook URL in Settings > Discord first. In Discord, go to Server Settings → Integrations → Webhooks → New Webhook, copy the URL, and paste it into WatchDog. Then right-click any clip and select "Share to Discord."
